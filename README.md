## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/buddhika2003/MyPortfolio.git
```

### Navigate to Project Directory

```bash
cd MyPortfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
